String.prototypes - Methods

The charAt() method returns the character at the specified index in a string.
The charCodeAt() method returns the Unicode of the character at the specified index in a string.
Also See: .charAt() | External Info: Unicode Table
The concat() method is used to join two or more arrays.
The endsWith() method determines whether a string ends with the characters of another string, returning true or false as appropriate.
Also See: .startsWith(), .includes(), .indexOf(), lastIndexOf()
The indexOf() method returns the position of the first occurrence of a specified value in a string. Case sensitive.
Also See: .lastIndexOf()
The lastIndexOf() method returns the position of the last occurrence of a specified value in a string. Case sensitive.
Also See: .indexOf()
The localeCompare() method compares two strings in the current locale. Returns -1 if str1 is sorted before str2. Returns 0 if the two strings are equal. Returns 1 if str1 is sorted after str2.

The match() method searches a string for a match against a regular expression, and returns the matches, as an Array object.
Also See:RegExp.prototype.exec(), RegExp.prototype.test()
The replace() method searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced.
Also See:String.prototype.match()
The search() method searches a string for a specified value, and returns the position of the match.
The slice() method extracts parts of a string and returns the extracted parts in a new string.
The split() method is used to split a string into an array of substrings, and returns the new array.
Also See: .charAt(), indexOf(), .join()
The substr() method extracts parts of a string, beginning at the character at the specified position, and returns the specified number of characters.
Also See: .substring()
The substring() method extracts the characters from a string, between two specified indices, and returns the new sub string.
Also See: .substr()
The toLocaleLowerCase() method returns the calling string value converted to lower case, according to any locale-specific case mappings.
Also See: .toLocaleUpperCase(), .toLowerCase()
The toLocaleUpperCase() method returns the calling string value converted to upper case, according to any locale-specific case mappings.
Also See: .toLocaleLowerCase(), .toUpperCase()
The toLowerCase() method converts a string to lowercase letters.
The toUpperCase() method converts a string to uppercase letters.
The toString() method returns the value of a String object.
The trim() method removes whitespace from both sides of a string.
The valueOf() method returns the primitive value of a String object.

String HTML Wrapper Methods

The anchor() method creates an <a> HTML anchor element that is used as a hypertext target.
Also See: .anchor()
The link() method is used to display a string as a hyperlink.
Also See: .anchor()
